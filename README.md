# 🌐 CyberSecure | Global Awareness Portal

> **"Unifying international intelligence to shield the digital frontier."**

CyberSecure is a state-of-the-art, futuristic web platform designed to educate the global population on the mechanics of cyber-crime. It combines high-impact visual design with interactive simulations to transform cybersecurity education into an immersive experience.

---

## ⚡ Core Systems

### 🧪 Simulation Lab
Step into the **Interactive Scam Simulation**. A choice-driven engine that tests your instincts against sophisticated "vishing" (voice phishing) attacks. Get real-time educational feedback on every decision you make.

### 🗺️ Global Threat Monitor
A live, SVG-powered visualization of worldwide security incidents. Integrated with a neon-alert ticker that broadcasts real-time cybersecurity protocols and global threat statuses.

### 📚 Intelligence Portal & Archive
Deep-dive into the **Global Incident Archive**. Analysis of high-profile real-world breaches like SolarWinds, WannaCry, and the Colonial Pipeline, detailing attack vectors, impact, and defensive lessons.

### 💾 Local Transmission Log
A functional mock-dataset powered by `localStorage`. Monitor your local data "transmissions," view saved history, and manage persistent information directly through a high-tech console interface.

---

## 🎨 Design Philosophy: "Cyber-Glassmorphism"
The portal uses a bespoke UI system designed to feel like a command center in the year 2050:
- **Glassmorphism**: Backdrop blurs and semi-transparent layers for depth.
- **Neon Highlights**: Primary cyan and secondary purple accents used for critical data visualization.
- **Micro-Animations**: Scroll-reveal effects and pulse animations for a dynamic, "living" interface.
- **Typography**: 'Source Code Pro' for that professional, terminal-inspired aesthetic.

---

## 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5, CSS3 (Custom Properties, Grid, Flexbox)
- **Logic**: Vanilla JavaScript (ES6+, DOM API)
- **Persistence**: Web Storage API (`localStorage`)
- **Fonts**: Google Fonts API
- **Visuals**: SVG Visualization & CSS Keyframe Animations

---

## 🚀 Deployment & Usage
Simply open `index.html` in any modern web browser to initialize the portal. No external dependencies or server-side requirements are necessary—the entire intelligence system runs client-side for maximum speed and privacy.
